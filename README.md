# Neural Cellular Automata

An interactive WebGL texture lab where simple local rules produce evolving organic patterns in real time. The app is built for fast iteration, live painting, and shader experimentation.

## Highlights

- GPU-driven neural cellular automata simulation
- Paint directly into the field with pointer input
- Live controls for mutation, speed, palette, and bloom
- Cloudflare Pages deployment flow

## Stack

- TypeScript
- WebGL2
- GLSL
- Vite

## Quick start

```bash
bun install
bun run dev
```

Build and deploy:

```bash
bun run build
bun run deploy
```
